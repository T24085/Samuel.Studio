# Samuel Studio Maximalist Starter

This is a static starter concept for a Samuel Studio homepage built around:
- full-screen storytelling sections
- Colombian maximalist editorial art direction
- still + motion presentation
- rich color, texture, and layered typography

## Included
- `index.html` — homepage starter
- `styles.css` — layout, color, and motion styling
- `script.js` — simple reveal and flash interactions
- `assets/` — placeholder SVG visuals you can replace with real photo/video assets
- `CODEX_HANDOFF.md` — concise brief for Codex / Cursor
- `mockup/wireframe.html` — presentation page showing the layout and section logic

## How to use
Open `index.html` locally in a browser, or upload the folder to GitHub Pages.

## What to replace first
1. Replace the SVG placeholders in `assets/` with real photography, motion posters, and video stills.
2. Add a real hero video file in the hero section.
3. Update the copy to match the final Samuel Studio positioning.
4. Upgrade transitions to GSAP / ScrollTrigger if you want a more premium effect.

## Suggested next build step
Convert this into a Vite + React + Tailwind project and split each panel into its own component.
