# Samuel Studio — Codex Handoff

Build a premium photography and motion website for **Samuel Studio**, positioned as a Colombian luxury image-making brand.

## Core direction
- Style: **maximalist experimental editorial**
- Mood: **high-end, cinematic, bold, sensual, art-directed**
- Cultural anchor: **very Colombian**, but never touristy or cliché
- Sections: every major section is **100vh full screen**
- Scroll behavior: each section should feel like a distinct scene
- Content mix: **still photography + motion/video**

## Brand sentence
Samuel Studio is a Colombian photography and motion company creating bold, cinematic visual systems for brands, people, and stories.

## Visual references to capture
- Colombian warmth, color, architecture, texture, rhythm, and density
- Editorial magazine layouts
- Fashion campaign art direction
- Layered poster composition
- Smooth but dramatic transitions between sections

## Avoid
- wedding-template softness
- generic agency layouts
- travel brochure vibes
- sterile minimalism
- overuse of gimmicky animation

## Recommended stack
- Vite
- React
- Tailwind CSS
- GSAP + ScrollTrigger for transitions
- Lenis or native smooth scroll if needed

## Palette
- Terracotta: `#C1442E`
- Marigold: `#F4A300`
- Cobalt: `#1C2C54`
- Tropical Green: `#1F7A63`
- Coral: `#FF6B5E`
- Black: `#0A0A0A`
- Cream: `#F5E9DA`

## Typography direction
- Elegant serif for headlines
- Clean modern sans for body and UI
- Oversized headlines with tight line-height
- Occasional uppercase tracking for editorial tension

## Transition system
Use only a few transition families and repeat them consistently:
- split reveal
- image expansion into next section
- layered parallax
- flash cut
- masked reveal

## Texture system
Apply carefully:
- film grain
- dust
- subtle light leaks
- paper edges / collage cuts
- plaster / wall texture

## Build notes
- sections must feel immersive, not like stacked cards
- images should overlap grid lines at times
- asymmetry is welcome, but keep it art-directed
- mobile version should preserve the mood without overloading animation
- use accessible contrast and keep nav simple
